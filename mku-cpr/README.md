# CPR Website (MKU Centre for Placenta Research)

This is the source code for the MKU Centre for Placenta Research (CPR) website.

## Tech Stack
- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Search**: [Pagefind](https://pagefind.app/)
- **Components**: React & Astro Components

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation
```bash
pnpm install
```

### Local Development
```bash
pnpm run dev
```

### Build for Production
```bash
pnpm run build
```

## Deployment
The site is built as a static site and can be deployed to any static host (GitHub Pages, Vercel, Netlify).

To deploy to GitHub Pages:
```bash
pnpm run deploy
```
